bikin web app untuk chat dengan AI berdasarkan topik.

AI menggunakan API gratis dari berbagai provider, tolong sediakan.
agent dan model bisa di customizable tergantung fitur yang diberikan oleh providernya.

gunakan nuxtjs v4 terbaru.
gunakan bootstrap v5 terbaru.
gunakan jquery jika memungkinkan.

agent bisa menggunakan provider dari ollama dan atau llamafile.

sitemap:
- topik
  - chat
- riwayat (sesi chat by topik)

dari sisi coding
- sediakan konfigrurasi untuk menentukan agent dan model
- sediakan konfigrurasi untuk menentukan topik
- setiap topik punya system prompt
