
## Local Server
- **DILARANG** menjalankan local server (seperti `php -S localhost:8000`).
- local server selalu menggunakan apache/httpd.
- local server sudah disediakan, untuk mengetahuinya URL nya, selalu cek `APP_URL` di `.env`.

## Environment & CLI
- proyek menggunakan PHP versi 8.3.
- untuk menjalankan perintah PHP gunakan `php83`.
- untuk menjalankan perintah composer gunakan `php83c`.
