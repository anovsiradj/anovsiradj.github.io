Example 5: Events & Custom Events

Demonstrates standard DOM events (click) and custom events with event bubbling and detail payloads.

Live Demo
How It Works
  • Standard Events: Click handlers bound to buttons
  • Custom Events: Component dispatches 'count-changed' event
  • Event Bubbling: Events bubble up and can be caught at document level
  • Event Detail: Custom data passed via event.detail

JavaScript - Standard Events:
on: {
  '.btn-inc': {
    click() {
      this.count++;
      this.updateDisplay();
      // Dispatch custom event
      this.dispatchEvent(new CustomEvent('count-changed', {
        bubbles: true, // Event bubbles up the DOM
        detail: { count: this.count, action: 'increment' }
      }));
    }
  },
  '.btn-dec': {
    click() {
      this.count--;
      this.updateDisplay();
      this.dispatchEvent(new CustomEvent('count-changed', {
        bubbles: true,
        detail: { count: this.count, action: 'decrement' }
      }));
    }
  },
  '.btn-reset': {
    click() {
      this.count = 0;
      this.updateDisplay();
      this.dispatchEvent(new CustomEvent('count-changed', {
        bubbles: true,
        detail: { count: this.count, action: 'reset' }
      }));
    }
  }
}
JavaScript - Listening for Custom Events:
// Listen at document level (event bubbling)
document.addEventListener('count-changed', (e) => {
  console.log('Count changed:', e.detail);
  // e.detail contains: { count, action }
});
